## 🎯 반응형 브레이크포인트

### CSS 미디어 쿼리

```css
/* 데스크탑 (≥1024px) */
.dashboard-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

/* 태블릿 (768px - 1023px) */
@media (max-width: 1023px) {
  .dashboard-grid {
    grid-template-columns: repeat(2, 1fr);
    gap: 15px;
  }
}

/* 모바일 (<768px) */
@media (max-width: 767px) {
  .dashboard-grid {
    grid-template-columns: 1fr;
    gap: 10px;
  }
}
```

## 🚀 인터랙션 플로우

### 사용 시나리오 1: 부모가 아이들 비교 보기

```
부모 로그인
  ↓
대시보드 허브 진입
  ↓
"📊 비교 대시보드" 클릭
  ↓
모든 아이들의 이번주 습관 달성률 시각화
  ↓
특정 아이 카드 클릭 (예: 영희)
  ↓
영희의 개별 대시보드로 이동
  ↓
영희의 기간별 추세, 자기인식, 월간 상세 정보 확인
```

### 사용 시나리오 2: 아이가 자신의 분석 보기

```
아이 로그인
  ↓
대시보드 허브
  ↓
"🧠 자기인식" 클릭
  ↓
약점/강점 분석, AI 제안 확인
  ↓
"다음주 목표 설정" 입력
  ↓
부모 알림 (아이가 목표를 설정했어요!)
  ↓
부모가 격려 메시지 작성
  ↓
아이 화면에 실시간 표시
```

### 사용 시나리오 3: 부모가 월간 리포트 생성

```
부모가 월간 상세 대시보드 진입
  ↓
"2025년 10월" 선택
  ↓
월간 통계, 주별 요약, 일별 히트맵 확인
  ↓
"PDF 다운로드" 클릭
  ↓
보고서 생성 & 다운로드
  ↓
학교/학원에 제출 가능
```
